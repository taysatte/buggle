# buggle 🪲 -- Daily Code Bug Challenge

### Description 🐛

<em><b>buggle</b></em> is a web application inspired by Wordle, designed to challenge
programmers with a daily code debugging puzzle. Each day, users are
presented with a short code snippet containing a hidden bug, and their
task is to identify and fix it.

This project aims to provide a fun and engaging way for developers to practice
their debugging skills and foster a sense of community.
